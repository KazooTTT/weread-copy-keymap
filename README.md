# 微信读书网页版复制快捷键

![cover](https://pictures.kazoottt.top/2024/06/20240605-f9f1fd7eec262178e50fb61695d12535.png)

这是一个油猴脚本，用于在微信读书的网页端新增复制快捷键。

在 windows 下，快捷键为`Ctrl + C`，在 mac 下，快捷键为`Cmd + C`。
